# Cricket Formats & Powerplays (Learning Dataset)

## T20
- 20 overs per innings
- Common powerplay: first 6 overs (fielding restrictions)

## ODI
- 50 overs per innings
- Common ODI phases (typical modern structure):
  - Overs 1–10: Powerplay
  - Overs 11–40: Middle overs
  - Overs 41–50: Death overs

## Test cricket
- Up to 5 days
- No fixed over limit per innings
- Draws are possible

## Strategy notes
- T20: higher run rate, more risk, more emphasis on boundaries
- ODI: balance between stability and acceleration
- Tests: patience, technique, managing sessions and pitch conditions
