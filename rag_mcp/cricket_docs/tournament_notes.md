# Cricket Tournaments — Notes (Learning Dataset)

This dataset is intentionally simplified and may not reflect real-world schedules.

## ICC events (general)
- ICC tournaments often include group stages followed by knockouts.

## Example tournament glossary
- **Group stage**: teams play multiple matches to qualify
- **Semi-final**: knockout match to reach final
- **Final**: championship match

## Domestic leagues (general)
- Franchise T20 leagues typically use:
  - League stage
  - Playoffs
  - Final

## Common points system (illustrative)
- Win: 2 points
- No result / tie (without super over): 1 point each
- Loss: 0 points
